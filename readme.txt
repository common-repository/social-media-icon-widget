A Wordpress widget that shows social media icons.

== Description ==

**Social Media Icon Widget**

This plugin shows your social media with icons. Add the widget to a widget position like the footer or sidebar, fill in the info and press save.
Your social media with icons is now being displayed on the front-end of your website!

**Features**

This plugin features:

* Twitter
* Facebook
* Google+
* Whatsapp
* Youtube
* Reddit
* Pinterest
* Skype
* Snapchat
* Github
* Instagram
* Linkedin
* Vine
* Tumblr
* Soundcloud
* Steam
* Vk
* Vimeo

**Important!**
You only have to add your username, don't paste a whole social media link.

== Installation ==

Upload the .zip file with a FTP program in the folder wp-content/plugins or upload via Worpress backend.

== Frequently Asked Questions ==

**My icons aren't showing, why?**

The social media icons can be blocked if you are using an ad-blocker. 
Disable the ad-blocker on the current page.

**How many icons are there?**

We currently have 18 social media icons.

**Where can I find the widget?**

On the backend of your website, go to appearance--> widget. Drag the social media widget into a widget position.
